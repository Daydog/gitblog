# [音标是发音基础元素](https://github.com/Daydog/gitblog/issues/8)

每个词都懂，组成一句就听不懂？ 看不懂？
音标是语言微观的解构，就像汉语的拼音和四声，你说话的时候不会感受它的存在，但是你小时候肯定学习过；你在生活中接触的是各种国家的英语变体，学习纯正的发音是为了辅助你更好的听懂各国口音
学英语选择什么材料？选择你感兴趣的任何材料比如电影游戏，记住学习是为了满足自己的好奇心，如果你没有兴趣，那么先停下来，找到自己兴趣所在；你正在从事的活动，可以选用英文资料去用，这样真正做到学以致用
远离社交媒体
列计划的时候绝对不要自我期望过高，比如一天学习三个，复习三个
用记事本来监控自己的学习流程
复习是学习的一部分，艾宾浩斯
立即行动吧
赖世雄提醒您，需要长期苦练

音节必须含有元音和辅音, 有几个元音，就有几个音节。英文是等音节语言，参见[英语的节奏和"等时性"(这才是中式英语的罪魁祸首!) - YouTube](https://www.youtube.com/watch?v=VMDhdaMkeBU)
重音 accent，双重音的有次重音
pronunciation 即下面的列表只解决一部分问题
intonation 即使元音、辅音、重音、语调都对了，还是不够
rhythm，节奏，重音规律。中文的节奏可以拆到每个字，每个字即为一个音节，音节等时
isochrony，等时性语言，英文是重音等时语言，这就是为什么你听到的外语是几个重音的连接
语言使用的习惯需要在环境中感受

- 辅音
    - web [w] 类似于汉字"乌"
    - yes [j] 类似于汉字"衣"
    - father [f] 类似于汉语拼音声母f
    - very [v] 发[f]时声带振动
    - red [r] 嘴巴向前撅, 舌头向后缩, 舌尖向上卷(r位于元音前)
    - car [r] 舌头向后缩, 舌尖向上卷(r位于元音后)
    - light [l] 舌尖抵住上齿背后, 把舌头滑落下来, 在滑落的过程中发音(l位于元音前)
    - well [l] 舌尖向前向上滑动, 抵住上牙齿背后, 在上滑的过程中发音(l位于元音后)
    - night [n] 和l动作一模一样, 但气流从鼻腔发出(n位于元音前)
    - pen [n] 和n动作一模一样, 但气流从鼻腔发出(n位于元音后)
    - mom [m] 类似于闭着嘴巴说"嗯"
    - sing [ŋ] 舌头后部上抬, 抵住软腭, 说"嗯"
    - roads [dz] 类似于汉字"资"
    - let's [ts] 类似于汉字"词"
    - boss [s] 类似于汉字"思"
    - rose [z] 发[s]时声带振动
    - thanks [ɵ] 舌尖放到上下齿之间, 然后发[s]
