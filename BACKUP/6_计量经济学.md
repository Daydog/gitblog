# [计量经济学](https://github.com/Daydog/gitblog/issues/6)

经济数据有截面数据（cross sectional data）、时序数据（time series data）和面板数据（panel data）三种类型，面板数据 即综合了前面两种
同一时点不同个体的差异叫组间差异 (between variation) ，不同时点同一个体的差异叫 组内差异(within variation)

# 静态短面板（static panel） 
如果解释变量（x）包含被解释变量的滞后值（yt-1），则称为“动态面板”（dynamic panel） 
静态指解释变量（x）不包含被解释变量（y）的滞后值，短面板指个体（n）多时期（T）少
对于截面数据就使用简单的线性回归，但是加入时序的维度后，就需要考虑个体效应（individual-specific effects model），个体效应是融入了无法观测的因素
## 回归元的种类 
- 可变回归元（Varying regressors）Xit，比如说 每一年的工资
- 不随时间改变的回归元（Time-invariant regressors）Xit=Xi for t，比如说你的性别
- 不随个体性质改变的回归元（Individual-invariant regressors），Xit=Xt for t，比如说当年国家的失业率，对于每个个体而言，那一年国家的失业率是相同的，不会因为你是沙雕你就会和其他人不一样 

# 回归模型（三种估计策略）
混合效应模型（Pooled model），也叫混合回归（Pooled regression），忽略个体效应，直接把所有数据混合到一起进行回归
固定效应模型（Fixed effect model）和随机效应模型（Random effect model）则考虑到了个体效应
用豪斯曼检验（Hausman，1978）可以判断到底使用FE还是RE
