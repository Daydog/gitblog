# [调节效应](https://github.com/Daydog/gitblog/issues/7)

调节变量（Moderator）是为了更完整的研究自变量 (P) 与因变量 (SR) 之间的 关系，可以是定性的 (如性别、种族、学校类型等) ,也可以是定量的(如年龄、受教育年限），我这里使用的 moderator BC 是定量数据，为了减少 dummy variable 的数量

当我们选择一个变量作为“中间变量”，这时候其实并不能确定它是“调节变量(Moderator) ” 还是 “中介变量(Mediator)”，它具体是哪种变量，会对 X 和 Y 之间的关系产生什么影响，是要根据数据分析的结果来判断
